Mosaico de Imágenes
Esta aplicación realizada en Python recrea imágenes utilizando otras imágenes más pequeñas. Tiene un sistema de detección de colores que permite elegir que imagen encaja mejor en cada sección de la fotografía original.
VER PROYECTO
Página Web Dinámica para el Club de Regatas Alte Brown
Este proyecto aún en desarrollo para Djando implica una herramientas fundamental para el club. No solamente habilita la posibilidad de llegar a un mayor público, sino que tambien permite gestionar socios con login, se utilizó un usuario custom creado a las necesidades indicadas por el cliente. Permite gestionar un sistema de turnos para inscripciones. Los administradores pueden obtener un listado de los inscriptos y mucho más.
