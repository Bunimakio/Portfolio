Agoodshop
Agoodshop es una aplicación realizada en puro HTML, CSS y Javascript. La misma realiza una petición a una API externa y se utiliza fetch para procesar los datos. La aplicación es dinámica, pues se adapta a la cantidad de datos que se carguen. Simula ser un carrito de compras donde el usuario puede seleccionar la cantidad de cada producto que desee y, a través del manejo del DOM, se actualiza dinamicamente la información. Se ha trabajado con programación orientada a objetos donde el objeto carrito es quien dispone de la información a procesar.
El programa lo que hará es leer desde una API, y tomará la información en formato JSON para volcarlo en los objetos y posteriormente renderizar la vista.
Posteriormente modificando el mismo JSON, agregando un producto, la vista se actualiza en consecuencia.
