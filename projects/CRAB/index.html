Club de Regatas Almirante Brown
La página web del Club Atlético Almirante Brown aún se encuentra en desarrollo. El proyecto nace como petición del club para lograr un mayor alcance y que sus socios puedan contar con un medio para estar al tanto de las novedades e información relevante. Tambien se buscaba lograr una automatización de los procesos de inscripción a distintos eventos para lograr un mejor registro y control.
Algunas funcionalidades
Sistema de usuarios:
La página cuenta con un sistema de logueo de usuarios aprovechando la potencia de listado herramientas brindadas por Django Auth, habiendo tres categorias: usuario no registrado, miembro y administrador. Cada uno cuenta con sus respectivos permisos. Los usuarios no registradon pueden visitar la página principal y ver las distintas noticias, sin embargo no pueden realizar inscripciones a actividades. Los miembros pueden inscribirse a las actividades y editar sus perfiles. Finalmente los administradores tienen la capacidad de agregar noticias y actividades, para ello la página cuenta con formularios de Django.
Actividades:
la necesidad de crear un sistema de control para las actividades nace debido al crecimiento del club, donde cada vez más miembros asistían a las remadas y los controles debían llevarse a mano, a través de planillas. Con este sistema los miembros registrados pueden ver una tabla donde los eventos se muestran listados, con sus respectivas observaciones, y cuentan con un botón de inscribirse. Tambien, en caso de necesitarlo, pueden desinscribirse. La página web generará un archivo PDF que los administradores pueden consultar con toda la información que el club a solicitado.
Noticias:
la creación de modelos de Django permite elaborar un sistema de noticias de facil uso, los administradores simplemente cargarán un formulario y se creara una "card" automaticamente dentro de un carrousel. Esto permite una gestión práctica, rápida y de facil uso.
Galeria:
La página utiliza un carrousel para mostrar como se desarrollan las actividades dentro del club, eventos y más.
